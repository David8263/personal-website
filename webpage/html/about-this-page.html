                        <script>
                            let previousDiv = null;
                            let previousA = null;
                            function toggleDiv(divId) {
                            const blank = document.getElementById('blank');
                            const div = document.getElementById(divId);
                                if (previousDiv && previousDiv !== div) {
                                    previousDiv.style.display = 'none';
                                }
                
                                if (div.style.display === "none") {
                                    blank.style.display = "none";
                                    div.style.display = "block";
                                    previousDiv = div;
                                } else {
                                    div.style.display = "none";
                                    blank.style.display = "block";
                                    previousDiv = null;
                                }
                            }
                            function updateActive(aId) {
                                const a = document.getElementById(aId);
                                if (previousA && previousA != a) {
                                    previousA.classList.remove('active');
                                }
                                a.classList.add('active');
                                previousA = a;
                                
                            }
                            
                            function projectTypeClicked(divId, aId) {
                                toggleDiv(divId);
                                if(previousA != document.getElementById(aId)) {
                                    updateActive(aId);
                                } else {
                                    const a = document.getElementById(aId);
                                    a.classList.remove('active');
                                    previousA = null;
                                }
                            }
                        </script>

As you can see on the right, A simple script tag is used to do two things, toggle which div is display: block (which on the projects page is the big yellow box) and which hyperlink has the active class. It accomplishes this by using global variables to store parameters from previous method calls and then compares the actively called values to previous ones.

This JavaScript actually caused me a lot of issues, though. I didn't quite understand the document.getElementById() that was used, but I somewhat understood the difference between passing an object and the value that I want to compare (from my previous Java Knowledge) so I knew that it needed to be there. I was having logic issues that seemingly had no basis to exit. This caused a really funny issue where, at first, I could go toggle on all of the project section highlights... but they couldn't turn off. Then I changed some code around and I could toggle each one on and off, but it didn't automatically disable the previous active highlight. Finally I came accross a solution... or so I thought. It properly toggled everything on and off except for one case, when you tried to toggle on one element, turn it off, and then turn that same one back on again. Thankfully though, I recognized quickly that this was because I forgot to update a variable, so it was a quick fix, but the strange logic that was happening was still a really cool pattern

Do you like all of the boxes? I created the system myself, but for the other part of my INF 286 final! It is a simple nested grid within a flexbox. There are two rows in the flexbox, one for content and one for the footer, and then the grid itself has two columns and as many rows as I need, depending on the number of divs I use. This allows me to neatly split of and display all of my content, and I'm really proud of the work I did! This is because, after my initial set up, it felt a lot like coding with bootstrap... except I created the library!
